# 📸 Responsive Image Gallery

A modern and responsive Image Gallery built using **HTML, CSS, and JavaScript**. The project provides an interactive user experience with category filtering, lightbox preview, image navigation, dark/light mode, image upload, and delete functionality.

---

## 🚀 Features

- 🖼️ Responsive Image Gallery
- 🔍 Filter Images by Categories
  - Nature
  - City
  - Animals
  - People
  - Miscellaneous
- 🌙 Dark Theme & ☀️ Light Theme
- 📷 Lightbox Image Preview
- ⬅️ Previous & ➡️ Next Navigation
- ➕ Add New Images
- 🗑️ Delete Images
- 📱 Mobile Friendly Responsive Design
- ⚡ Smooth Hover Effects

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome Icons
